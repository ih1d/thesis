\section{Introduction}

This technical report introduces El MegaProbe Language (MPL), a statically-typed
functional language with ML-like syntax that has first-class bioinformatics and 
bioinformatics primitives. 

The argument for functional languages has been widely studied~\cite{whyfp}~\cite{totalfp}, hence,
the reason for developing a lazy, functional language. 

\section{Methods}

Building the language follows the convention from Krishnamurthi~\cite{plai}. 
The language used to implement was Haskell~\cite{haskell} since it is a lazy, purely functional
PL\@. Parser combinators~\cite{parser} were used for lexing and parsing. Backus-Naur Form (BNF)
was used to design the syntax of the language.

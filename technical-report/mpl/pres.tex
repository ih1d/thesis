\begin{frame}
\frametitle{Introduction}
\begin{itemize}
    \item El MegaProbe Language: Statically-typed, ML-like language with bioinformatics primitives
    \item Help The Blueprint of Life project with a language that can handle large genome sequences
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Methods}
The language implementation follows the convention from Krishnamurthi~\cite{plai}.
\begin{itemize}
    \item Lexing and parsing: parser combinators~\cite{parser}
    \item Type Checking and Evaluating: Pattern Matching
\end{itemize}
\end{frame}


\begin{frame}
\frametitle{Current Results}
\begin{itemize}
    \item An interpreter
    \item DNA and RNA sequences as primitives, and higher-order functions like transcribe DNA sequence and reverse complement.
    \item (minimal) Apache Arrow bindings for fast reading csv files and producing Apache Arrow Tables.
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Conclusions}
A fully functional interpreter has been built with most common programming constructs and primitives, with additional bioinformatics
primitives and functions.

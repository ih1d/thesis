\begin{frame}
\frametitle{Methods}
The language implementation follows the convention from Krishnamurthi~\cite{plai}.
\begin{itemize}
    \item Lexing and parsing: parser combinators~\cite{parser}
    \item Type Checking and Evaluating: Pattern Matching
    \item Dataframe: Apache Arrow API
\end{itemize}
